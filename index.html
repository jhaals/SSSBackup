SSSBackup by ScriptDev

Simple Secure System Backup uses well-tested techniques for creating tar.bz2 archives of selected folder and transfer them encrypted to you're remote machine. SSSBackup also compare you're archive so you got the same data on both machines. The results are then sent to you by email if you wish.

Dependencies

python 2.3, scp
